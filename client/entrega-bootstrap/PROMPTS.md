# Prompts (Entrega Bootstrap)

## Prompt 1

Genera una pantalla de Login en HTML + Bootstrap 5. Debe ser responsive, con card centrada, inputs correo/contraseña y botón.

## Prompt 2

Genera una pantalla de Registro en HTML + Bootstrap 5. Responsive, inputs nombre/correo/contraseña y botón.

## Prompt 3

Genera un Dashboard en HTML + Bootstrap 5 con navbar, sidebar y cards responsivas de resumen.

## Prompt 4

Genera una pantalla de listado con tabla Bootstrap (responsive) y un modal Bootstrap para ver el detalle. Datos hardcode.

## Prompt 5

Genera una pantalla de formulario Bootstrap para crear/editar una reserva con grid (servicio, fecha, hora, cliente, estatus, notas).
